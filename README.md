# Sales & Revenue Analysis Dashboard

## Project Overview
This project involves a comprehensive Sales and Revenue Analysis Dashboard built using Python and Excel. It transforms raw transactional data into actionable business insights through automated data processing and visualization.

## Key Features
- **KPI Tracking:** Real-time calculation of Total Revenue, Profit, and Average Order Value (AOV).
- **Trend Analysis:** Monthly revenue forecasting and historical trend visualization.
- **Product Performance:** Identification of top-selling products and categories.
- **Geographic Insights:** Sales distribution across different global regions.

## Technologies Used
- **Python:** Data cleaning and manipulation.
- **Pandas:** For handling large datasets and aggregations.
- **Openpyxl:** To generate the interactive Excel dashboard and automated charting.

## How to Use
1. Download the `Sales_Revenue_Analysis_Dashboard.xlsx` file.
2. View the `Dashboard` tab for the visual summary.
3. Use the `Raw_Sales_Data` tab to add new transactions; the summary logic is designed to scale.

## Insights Generated
- Identified seasonal peaks in consumer electronics during Q2.
- The **ProPhone X** accounts for the highest share of total revenue.
- Consistent growth observed in the 'Wearables' category across European markets.
